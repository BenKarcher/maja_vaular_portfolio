DIGITAL MENU

This was a school assignment where we used another students design manual to create a digital menu prototype for tablets. The student who made the design manual is Liv Wallace. She made the logo and chose the colors and typography used in the project.

PLAYFUL, COLORFUL AND USER FRIENDLY

The restaurant has a playful image and is also a child-friendly enviourment, so I kept that in mind when I designed the menu. The fact that children will use the menu meant it would have to be very user friendly, so I made it as simple to use as possible..
